# 🚗 BestCars - Sistema de Gestão de Veículos


## 🚀 Sobre o Projeto

O **BestCar** é um sistema web desenvolvido para exibição e gestão de veículos, com foco em organização, apresentação visual e experiência do usuário.

O projeto simula uma plataforma automotiva moderna, permitindo visualizar carros, detalhes técnicos e informações relevantes de forma clara e atrativa.

---

## 🎯 Objetivo

Criar uma interface moderna para:

- Exibição de veículos
- Organização de informações automotivas
- Apresentação de detalhes técnicos
- Experiência visual atrativa

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Design Responsivo

---

## ⚙️ Funcionalidades

- 🚗 Listagem de veículos
- 📊 Exibição de informações (modelo, preço, detalhes)
- 🖼️ Galeria de imagens
- 📱 Layout responsivo (mobile e desktop)
- 🔍 Organização visual intuitiva
- 🎯 Destaque para veículos principais
